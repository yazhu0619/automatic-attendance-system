# Automatic Attendance System

An automated solution designed to log and manage attendance seamlessly using a web dashboard interface and front-end logic.

## Features
* **Interactive Dashboard:** Clean, responsive web UI for tracking status.
* **Real-time Scans:** Simulated camera triggering and instant status updates.
* **Lightweight Stack:** Built using native web technologies for high performance.

##  Tech Stack
* **HTML5:** Project structure and layout skeleton.
* **CSS3:** Custom modern UI theme and button transitions.
* **JavaScript (ES6):** Dynamic event listeners and simulated scanning logic.
